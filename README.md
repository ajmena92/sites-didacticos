# Sites Didácticos

Colección de materiales didácticos para **Python**, **Linux** y **CCNA**, publicada como GitHub Pages.

## Estructura

```
/
├── index.html              ← Página principal
├── assets/css/style.css    ← Estilos globales
├── python/index.html       ← Sección Python
├── linux/index.html        ← Sección Linux
└── ccna/
    ├── index.html          ← Sección CCNA
    └── ccna1/
        └── subnetting.html ← Subnetting y VLSM
```

## GitHub Pages

El sitio se publica desde la rama `main`, directorio raíz (`/`).
Configura en: **Settings → Pages → Source: main / (root)**
